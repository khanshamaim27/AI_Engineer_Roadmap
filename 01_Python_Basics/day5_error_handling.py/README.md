# 🐞 Day 05 - Python Error Handling Challenge

Welcome to the Day 05 Debugging Challenge!

This project is designed for beginners learning Python error handling.

Your goal is to find and fix the bugs without looking at the solution first.

## Topics Covered

* try
* except
* else
* finally
* raise
* ValueError
* ZeroDivisionError
* IndexError
* TypeError
* Debugging with print()

## Challenges

Fix the following bugs:

1. Division by zero
2. Invalid integer input
3. List index out of range
4. Wrong data type
5. Custom exception using `raise`

After fixing all the bugs, compare your solution with `solution.py`.

Happy Coding!
